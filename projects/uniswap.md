# Uniswap on Base
Category: DeFi  
Type: Automated Market Maker (AMM)
Overview:
Uniswap is a decentralized exchange protocol enabling permissionless token swaps using liquidity pools.
Deployment on Base allows lower fees and improved transaction speed.
## Core Mechanism
Uniswap uses constant product market making (x * y = k).
Liquidity providers deposit token pairs and earn fees from swaps.
## Strategic Role on Base
Uniswap provides foundational liquidity and supports token launches.
Its presence increases ecosystem credibility.
## Risk Factors
- Liquidity migration risk
- Governance concentration
- Smart contract vulnerabilities
## Fee Structure
Liquidity providers earn swap fees.
Fee tiers influence capital allocation efficiency.
## Ecosystem Integrations
Uniswap integrates with wallets, analytics dashboards, and aggregators.
