# Aerodrome Finance
Category: DeFi  
Type: Decentralized Exchange (DEX)
Overview:
Aerodrome is a liquidity protocol designed to facilitate token swaps on Base.
Core Features:
- Liquidity pools
- Token swapping
- Yield generation for LPs
## Value Proposition
Aerodrome aims to concentrate liquidity and provide efficient trading routes.
It positions itself as a central liquidity layer within the Base ecosystem.
